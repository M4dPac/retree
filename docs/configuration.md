# ⚙️ Конфигурация

> В текущей версии конфигурационный файл не поддерживается.  
> Планируется поддержка `~/.rtreerc.toml` в будущих релизах.

---

## Переменные окружения

| Переменная    | Описание                                               |
| ------------- | ------------------------------------------------------ |
| `TREE_COLORS` | Настройка цветов (приоритет над `LS_COLORS`)           |
| `LS_COLORS`   | Настройка цветов в формате GNU ls                      |
| `TREE_LANG`   | Язык интерфейса (`en` или `ru`)                        |
| `NO_COLOR`    | Отключает все цвета, если установлена (любое значение) |
