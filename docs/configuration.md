# ⚙️ Конфигурация

> Конфигурационный файл в текущей версии не поддерживается.  
> Поддержка `~/.rtreerc.toml` запланирована в будущих релизах.

---

## Переменные окружения

| Переменная    | Значения       | Описание                                     |
| ------------- | -------------- | -------------------------------------------- |
| `TREE_COLORS` | строка цветов  | Настройка цветов (приоритет над `LS_COLORS`) |
| `LS_COLORS`   | строка цветов  | Настройка цветов в формате GNU ls            |
| `TREE_LANG`   | `en` или `ru`  | Язык интерфейса (то же, что `--lang`)        |
| `NO_COLOR`    | любое значение | Отключает все цвета при наличии переменной   |

### Установка переменных

**Linux / macOS:**

```bash
export TREE_LANG=ru
export TREE_COLORS="di=1;34:ex=1;32:*.rs=1;33"
export NO_COLOR=1
```

**Windows PowerShell:**

```powershell
$env:TREE_LANG = "ru"
$env:TREE_COLORS = "di=1;34:ex=1;32:*.rs=1;33"
$env:NO_COLOR = "1"
```

**Windows cmd:**

```cmd
set TREE_LANG=ru
set TREE_COLORS=di=1;34:ex=1;32:*.rs=1;33
```

---

## Приоритет настроек

Флаги командной строки всегда перекрывают переменные окружения.

```
флаги CLI  >  TREE_COLORS / TREE_LANG  >  LS_COLORS  >  значения по умолчанию
```
