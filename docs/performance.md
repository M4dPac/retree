# ⚡ Производительность

rtree оптимизирован для эффективной обработки больших деревьев каталогов.

---

## Параллельный режим
```powershell
# Авто-определение потоков
rtree --parallel

# Явное указание количества потоков
rtree --parallel --threads 4
```

Параллельный режим использует [Rayon](https://github.com/rayon-rs/rayon) с алгоритмом work-stealing для балансировки нагрузки.

---

## Бенчмарки

**Запуск всех бенчмарков:**
```powershell
cargo bench
```

**Сохранение baseline:**
```powershell
cargo bench -- --save-baseline main
```

**Сравнение с baseline:**
```powershell
cargo bench -- --baseline main
```

> Измерение: median time (Criterion) · Режим: `release`
