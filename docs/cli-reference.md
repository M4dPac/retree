# 📖 CLI Reference

```
rtree [OPTIONS] [PATH...]
```

Если путь не указан, используется текущий каталог. Можно указать несколько путей для отображения нескольких деревьев.

---

## 🖥️ Параметры отображения

| Флаг   | Полная форма      | Описание                                      |
| ------ | ----------------- | --------------------------------------------- |
| `-a`   | `--all`           | Показать скрытые файлы                        |
| `-d`   | `--dirs-only`     | Только каталоги                               |
| `-l`   | `--follow`        | Следовать по символическим ссылкам            |
| `-f`   | `--full-path`     | Показывать полный путь                        |
| `-x`   | `--one-fs`        | Не выходить за пределы ФС                     |
| `-L N` | `--level N`       | Ограничить глубину                            |
|        | `--filelimit N`   | Пропускать каталоги с > N файлов              |
|        | `--noreport`      | Не выводить итог                              |
|        | `--parallel`      | Параллельный обход                            |
|        | `--threads N`     | Количество потоков                            |
|        | `--queue-cap N`   | Ёмкость очереди на поток (по умолчанию: 4096) |
|        | `--streaming`     | Потоковый вывод (рендер во время обхода)      |
|        | `--max-entries N` | Ограничить общее число записей                |

> **Примечание:** `--streaming` применяется только к текстовому выводу. С `--prune` или структурными форматами (`-J`, `-X`, `-H`) автоматически используется стандартный обход.

## 🔍 Фильтрация

| Флаг         | Полная форма        | Описание                        |
| ------------ | ------------------- | ------------------------------- |
| `-P PATTERN` | `--pattern PATTERN` | Показывать только файлы по glob |
| `-I PATTERN` | `--exclude PATTERN` | Исключить файлы по glob         |
|              | `--matchdirs`       | Применять шаблоны к каталогам   |
|              | `--ignore-case`     | Поиск без учёта регистра        |
|              | `--prune`           | Не показывать пустые каталоги   |

### Glob-синтаксис

| Символ   | Описание                          |
| -------- | --------------------------------- |
| `*`      | Любая последовательность символов |
| `?`      | Любой одиночный символ            |
| `[abc]`  | Любой символ из скобок            |
| `[a-z]`  | Любой символ из диапазона         |
| `[!abc]` | Любой символ НЕ из скобок         |

## 🔢 Сортировка

| Флаг          | Полная форма     | Описание                                            |
| ------------- | ---------------- | --------------------------------------------------- |
| `-v`          | `--version-sort` | Натуральная сортировка (`file2` перед `file10`)     |
| `-t`          | `--timesort`     | Сортировка по времени модификации                   |
| `-c`          | `--ctime`        | Сортировка по времени изменения метаданных          |
| `-U`          | `--unsorted`     | Без сортировки                                      |
| `-r`          | `--reverse`      | Обратный порядок сортировки                         |
|               | `--dirsfirst`    | Каталоги первыми                                    |
|               | `--filesfirst`   | Файлы первыми                                       |
| `--sort TYPE` |                  | `name`, `size`, `mtime`, `ctime`, `version`, `none` |

| -------------- | ---------------- | ------------------------- |

| `-n`           | `--nocolor`      | Без цвета                 |
| `-C`           | `--color-always` | Цвет всегда               |
| `--color WHEN` |                  | `auto`, `always`, `never` |

## 📄 Метаданные

| Флаг            | Полная форма | Описание                                                           |
| --------------- | ------------ | ------------------------------------------------------------------ |
| `-s`            | `--size`     | Размер в байтах                                                    |
| `-h`            | `--human`    | Человекочитаемый формат (`1K`, `2M`, `3G`)                         |
|                 | `--si`       | Единицы СИ (степени 1000)                                          |
| `-D`            | `--date`     | Дата модификации                                                   |
| `--timefmt FMT` |              | Формат даты (синтаксис `strftime`), по умолчанию: `%Y-%m-%d %H:%M` |
| `-p`            | `--perm`     | Права доступа / атрибуты                                           |
| `-u`            | `--uid`      | Владелец файла                                                     |
| `-g`            | `--gid`      | Группа файла                                                       |
|                 | `--inodes`   | Номера inode                                                       |
|                 | `--device`   | Номера устройств                                                   |
| `-F`            | `--classify` | Добавлять `/` для каталогов, `@` для ссылок, `*` для исполняемых   |
| `-q`            | `--safe`     | Заменять непечатаемые символы на `?`                               |
| `-N`            | `--literal`  | Выводить непечатаемые символы как есть                             |
| `--charset ENC` |              | Кодировка вывода (например, `utf-8`)                               |

## 📤 Экспорт

| Флаг       | Полная форма    | Описание                        |
| ---------- | --------------- | ------------------------------- |
| `-o FILE`  | `--output FILE` | Вывод в файл                    |
| `-J`       | `--json`        | JSON-вывод                      |
|            | `--json-pretty` | Форматированный JSON-вывод      |
| `-X`       | `--xml`         | XML-вывод                       |
| `-H URL`   | `--html URL`    | HTML-вывод с базовым URL        |
| `-T TITLE` | `--title TITLE` | Заголовок HTML                  |
|            | `--nolinks`     | Не создавать гиперссылки в HTML |
|            | `--hintro FILE` | Свой HTML-заголовок из файла    |
|            | `--houtro FILE` | Своё HTML-окончание из файла    |
