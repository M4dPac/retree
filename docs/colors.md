# 🎨 Настройка цветов

rtree поддерживает следующие переменные окружения:

- `LS_COLORS` — стандартный формат GNU ls
- `TREE_COLORS` — приоритетнее `LS_COLORS`
- `NO_COLOR` — отключает все цвета (при любом значении)

---

## Пример

**Linux / macOS:**

```bash
export LS_COLORS="di=1;34:*.rs=0;33"
export TREE_COLORS="di=1;34:ex=1;32:*.rs=1;33"
```

**Windows PowerShell:**

```powershell
$env:LS_COLORS = "di=1;34:*.rs=0;33"
$env:TREE_COLORS = "di=1;34:ex=1;32:*.rs=1;33"
```

---

## Формат записи

```
TYPE=STYLE;FG;BG
```

Несколько типов разделяются двоеточием: `di=1;34:*.rs=0;33`.

### Типы

| Код     | Описание             |
| ------- | -------------------- |
| `di`    | Каталог              |
| `fi`    | Обычный файл         |
| `ln`    | Символическая ссылка |
| `or`    | Битая ссылка         |
| `ex`    | Исполняемый файл     |
| `hi`    | Скрытый (Windows)    |
| `sy`    | Системный (Windows)  |
| `*.ext` | Файлы по расширению  |

### Стили

| Код | Стиль        |
| --- | ------------ |
| `0` | Обычный      |
| `1` | Жирный       |
| `2` | Тусклый      |
| `3` | Курсив       |
| `4` | Подчёркнутый |

---

## Цвета

**Стандартные** (`30–37`) и **яркие** (`90–97`):

| Код  | Цвет      | Яркий |
| ---- | --------- | ----- |
| `30` | Чёрный    | `90`  |
| `31` | Красный   | `91`  |
| `32` | Зелёный   | `92`  |
| `33` | Жёлтый    | `93`  |
| `34` | Синий     | `94`  |
| `35` | Пурпурный | `95`  |
| `36` | Голубой   | `96`  |
| `37` | Белый     | `97`  |

**256 цветов:**

```
38;5;N        # передний план
48;5;N        # задний план
```

**True Color (24-бит):**

```
38;2;R;G;B    # передний план
48;2;R;G;B    # задний план
```

## Цвета по умолчанию

```
di=1;34       # Каталоги: жирный синий
ln=1;36       # Ссылки: жирный голубой
or=1;31;40    # Битые ссылки: жирный красный на чёрном
ex=1;32       # Исполняемые: жирный зелёный
fi=0          # Обычные файлы: терминальный цвет по умолчанию
hi=2;37       # Скрытые (Windows): тусклый белый
sy=2;37       # Системные (Windows): тусклый белый

*.rs=0;33     *.py=0;33     *.js=0;33
*.zip=1;33    *.tar=1;33    *.gz=1;33
*.png=1;35    *.jpg=1;35    *.gif=1;35
*.exe=1;32    *.bat=1;32    *.ps1=1;32
```
